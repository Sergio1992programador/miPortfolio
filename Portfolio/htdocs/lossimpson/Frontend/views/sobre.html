El Proyecto

Un homenaje digital a Los Simpson

Este sitio web es un espacio interactivo donde los fans de Los Simpson pueden descubrir y disfrutar contenido inspirado en la serie. Combina diseño moderno con referencias nostálgicas, tipografías y colores inspirados en Springfield. Es también un portafolio vivo que refleja mi estilo y forma de trabajar.

Tecnologías Utilizadas

Frontend y Backend

HTML5 y CSS3 para estructura y estilo. JavaScript para interactividad. Bootstrap para diseño responsive. PHP para lógica del servidor. MySQL para gestión de bases de datos. Todo esto asegura que la página sea rápida, funcional y escalable.

Visión

Diseño con propósito

Mi visión como diseñador web es crear proyectos que transmitan emociones y conecten con las personas. Los Simpson representan humor y cultura popular, y he querido que todo eso se refleje en esta página. Cada detalle está pensado para ofrecer una experiencia inmersiva y entretenida.
